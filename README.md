* 在临摹网页的过程中，出现频次最高的有导航、icon、form表单（搜索框
、按钮等等）、列表、分列式结构、响应式布局、基础点击切换事件的绑定，还有很多我还没有开始涉及到，但是从中也收获颇丰。

* 对于background属性，其中的背景图片，涉及到sprite（雪碧图），理解也深刻了，position的设定，width，height的设定，overflow，z-index等等。
